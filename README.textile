h1. Color Theme Dirac

p. A pastel color theme for GNU Emacs, somewath inspired by "color-theme-galois":http://tuxlife.altervista.org/wordpress/?p=560 .

h2. Usage

p. After cloning the repository with %{font-family: fixed; color: blue; background: "light gray"}git clone git://github.com/nicodds/color-theme-dirac.git%, you could use the theme by writing these lines of elisp in your %{font-family: fixed; color: blue; background: "light gray"}~/.emacs% file:

p{font-family: fixed; color: blue; background: "light gray"}.(require 'color-theme)<br />(load-file "~/path_chosen/color-theme-dirac/color-theme-dirac.el")<br />(color-theme-dirac)
